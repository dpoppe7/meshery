<h1>Infrastructure Management Guides</h1>

Guides for understanding Meshery's ability to manage infrastructure.

- [Deploying Sample Applications](/guides/infrastructure-management/deploying-sample-apps) - This guide is to help users get a better understanding of sample apps
- [Managing Events with Notification Center](/guides/infrastructure-management/events-management) - Meshery tracks operations performed on your infrastructure and workloads, and provides notification of environment issues, application conflicts with infrastructure configuration, policy violations, and so on.
- [Envoy WASM Filter Management](infrastructure-management/filter-management) - Meshery provides the ability to manage WASM filters for data plane proxies for agility, maintainability, diversity, reliability and isolation, security, and speed.
- [GitOps with Meshery](/guides/infrastructure-management/gitops-with-meshery) - Integrating your CI/CD pipelines with Meshery's GitHub Actions
- [Infrastructure Discovery](infrastructure-management/infrastructure-discovery) - MeshSync supports both greenfield and brownfield discovery of infrastructure. Greenfield discovery manages infrastructure created and managed entirely by Meshery, while brownfield discovery identifies separately created infrastructure.
- [Infrastructure Management](guides/infrastructure-management/overview) - Meshery provides the ability to manage infrastructure for agility, maintainability, diversity, reliability and isolation, security, and speed.
- [Managing Connections](infrastructure-management/lifecycle-management) - Manage the lifecycle of your infrastructure by registering each infrastructure element with Meshery.
